#  Análise de Dados de uma Livraria com SQL

## Sobre o projeto

Este projeto foi desenvolvido para analisar um banco de dados de uma livraria utilizando **SQL**.

O objetivo é responder perguntas de negócio por meio de consultas SQL e extrair insights sobre livros, autores, editoras, avaliações e comportamento dos usuários.

---

## Objetivo

Realizar consultas SQL para responder questões de negócio, como:

- Quantos livros foram publicados após 1º de janeiro de 2000?
- Qual editora publicou mais livros?
- Quais autores possuem as melhores avaliações?
- Quais livros receberam mais avaliações?
- Como os usuários interagem com a plataforma?
